# year

Небольшое веб-приложение для проверки, является ли год високосным.

Функциональность
- Проверка одного года (в веб-форме или через API).
- Проверка списка годов.
- Нахождение предыдущего и следующего високосного года для заданного года.

Файлы проекта
- `leap_year.py` — основная логика: `is_leap_year`, `next_leap_year`, `previous_leap_year`.
- `app.py` — Flask-приложение с API и веб-интерфейсом.
- `templates/index.html` — фронтенд страницы.
- `static/` — стили и скрипты.
- `tests/` — юнит- и интеграционные тесты.

Зависимости
- Python 3.10+ (или совместимая версия)
- Flask

Установка
1. Создайте виртуальное окружение (рекомендуется):

```bash
python -m venv .venv
source .venv/bin/activate
```

2. Установите зависимости:

```bash
pip install -r requirements.txt
```

Запуск приложения
1. Запустить сервер в режиме разработки:

```bash
cd /path/to/project
python app.py
```

2. Откройте в браузере (локально):

```
http://127.0.0.1:5000/
```

API
- `GET /api/check?year=2024` — проверяет год, возвращает JSON.
- `POST /api/check` с JSON `{"year": "2024"}` — альтернативный способ.
- `POST /api/check-multiple` с JSON `{"years": ["2023", "2024"]}` — проверка нескольких лет.
- `GET /api/adjacent-leap-years?year=2023` — возвращает предыдущий и следующий високосный годы.

Тесты
1. Установите `pytest` (если ещё не установлен):

```bash
pip install pytest
```

2. Запустите тесты:

```bash
pytest -q
```

Примечания
- Сервер запущен в режиме разработки (Flask dev server). Для продакшена используйте WSGI-сервер (gunicorn, uWSGI) и настройте reverse-proxy.
- Порт по умолчанию: `5000`.

Автор: Этот репозиторий создан как демонстрация простого веб-приложения для проверки високосных лет.
